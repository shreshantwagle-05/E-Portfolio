Section 01 — Overview

What is Full Stack Development?

Full stack development covers every layer of a web application — the visual interface users see, the server logic that processes data, and the database that stores it permanently.

Frontend
Backend
Database
APIs
DevOps
🖥️
Frontend

What users see and touch. Built with HTML, CSS, JavaScript and Bootstrap.

⚙️
Backend

Server-side logic. Handles requests, authentication, and business rules.

🗄️
Database

Stores all data permanently. MySQL for structured, MongoDB for flexible.

How a Food Order Works

When a customer places an order on a delivery app, the request travels through every layer of the full stack in under two seconds. Here is exactly what happens at each step.

1
Customer Taps "Place Order"

JavaScript captures the click event in the browser — no page reload

2
Frontend Sends the Request

Cart data is packaged as JSON and sent via HTTP POST to the backend API

3
Backend Processes Everything

Validates input, checks authentication, processes payment via Stripe

4
Database Saves the Order

New order record written with ID, items, price, address, and status

✓
Confirmation Shown to Customer

Success response sent back — frontend updates the page instantly

Section 02 — Markup

HTML – Structure of Every Page

HTML (HyperText Markup Language) is the skeleton of every webpage. It uses tags to label content so the browser knows what each piece is and how to display it.

📌
<h1>–<h6>

Heading hierarchy for page structure

🔗
<a>

Anchor links — connects pages together

📝
<form>

Collects user input for the backend

🖼️
<img>

Embeds images with alt text for accessibility

index.html 
<!-- Standard HTML5 document structure -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My ePortfolio</title>
</head>
<body>

  <h1>Welcome to My Portfolio</h1>
  <p>CET138 Full Stack Development student.</p>

  <!-- Unordered list of skills -->
  <ul>
    <li>HTML & CSS</li>
    <li>Bootstrap Framework</li>
    <li>JavaScript</li>
  </ul>

  <!-- Image with alt text for accessibility -->
  <img src="profile.jpg"
       alt="Profile photo">

  <!-- Contact form -->
  <form>
    <input type="text"
           placeholder="Your name">
    <button>Send Message</button>
  </form>

</body>
</html>
Section 03 — Framework

Bootstrap – Build Faster

Bootstrap is a free CSS framework that provides a 12-column responsive grid and ready-made components — navbar, cards, buttons — all with a single CDN link.

index.html — Bootstrap grid

Section 04 — Interactivity

JavaScript – Bringing Pages to Life

JavaScript is the only language that runs in the browser and makes pages respond to what the user does — updating content, validating forms, and communicating with servers.

app.js 
// HTML elements found by ID
const correctBtn = document.getElementById("correct-btn");
const wrongBtn   = document.getElementById("wrong-btn");
const feedback   = document.getElementById("quiz-feedback");

// Event listener — waits for a click
correctBtn.addEventListener("click", function() {
  feedback.textContent = "✓ Correct! Well done.";
  feedback.className = "quiz-feedback-box correct-fb show";
  correctBtn.classList.add("correct");
});

wrongBtn.addEventListener("click", function() {
  feedback.textContent = "✗ Not quite. Try again!";
  feedback.className = "quiz-feedback-box wrong-fb show";
  wrongBtn.classList.add("wrong");
});

// Reusable function
function greetUser(name) {
  return "Hello, " + name + "!";
}
console.log(greetUser("Dipesh"));
