About This Portfolio

This ePortfolio was created as part of the CET138 Full Stack Development module. It covers the core technologies used to build modern websites — HTML for structure, CSS for styling, Bootstrap for responsive layouts, and JavaScript for interactivity. Each section includes a written explanation, a code example, and a live working demo.

What This Portfolio Demonstrates

Every section is built from original, working code that matches the written report for CET138.

📄
HTML Structure and Semantic Elements

Headings, paragraphs, lists, links, images, and forms — the building blocks of every webpage. A complete personal profile page is built and explained step by step.

🎨
CSS Styling, Box Model and Flexbox

The same profile page is then styled using an external CSS file — covering selectors, colours, spacing, and layout with Flexbox. The live result updates on screen.

⚡
Bootstrap Grid and Responsive Components

A restaurant homepage ("The Good Plate") is built using only Bootstrap class names — demonstrating the 12-column grid, navbar, cards, and buttons.

🔧
JavaScript DOM Manipulation and Events

A simple interactive quiz asks "What does HTML stand for?" and gives instant correct/wrong feedback — demonstrating variables, event listeners, and DOM updates without any page reload.

🌐
Full Stack Architecture and Data Flow

How the browser, server, and database work as a system. The journey of a request through all layers of the stack is traced and explained with clear diagrams.

📝
Reflections on Learning and Connections

Each section ends with a personal reflection — what was surprising, what clicked, common mistakes, and how each technology connects to the others and to real full stack projects.
